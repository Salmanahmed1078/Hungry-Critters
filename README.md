# Hungry Critters

A 2D browser game where you match food to hungry critters before they lose patience. Fast reactions and pattern recognition win the day.

## How to Play

- Critters appear on screen each showing what food they want
- Click or drag the correct food to the right critter
- Match all critters before the timer runs out
- Wrong matches cost you time
- Score multipliers reward quick, accurate matches

## Features

- Multiple critter and food types
- Timer-based rounds
- Score multiplier system
- Progressive difficulty
- Animated sprites

## Getting Started

```bash
git clone https://github.com/Salmanahmed1078/Hungry-Critters.git
cd Hungry-Critters
# Open index.html in your browser
```
